* `Ctrl+w` + `+/-` : increase/dicrease height (ex. `20<C-w>+`)
* `Ctrl+w` + `>/<` : increase/dicrease width (ex. `20<C-w><`)
* `Ctrl+w` + `_` : set height (ex. `50<C-w>_`)
* `Ctrl+w` + `|` : set width (ex. `50<C-w>|`)
* `Ctrl+w` + `=` : equalize w and h
